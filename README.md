# Expense Tracker — Трекер расходов

**Автор:** Евгения Кибалина

## Описание программы

Expense Tracker — это графическое приложение для учёта личных финансов. Программа позволяет добавлять расходы, фильтровать их по категориям и датам, а также автоматически подсчитывать сумму трат за выбранный период. Все данные сохраняются в JSON-файл и загружаются при следующем запуске.

### Основные возможности:
- Добавление расходов с указанием суммы, категории и даты
- Автоматическая валидация вводимых данных
- Фильтрация расходов по категории
- Фильтрация расходов по диапазону дат
- Подсчёт общей суммы расходов за отфильтрованный период
- Удаление записей
- Сохранение и загрузка данных в формате JSON
- Удобный табличный интерфейс
